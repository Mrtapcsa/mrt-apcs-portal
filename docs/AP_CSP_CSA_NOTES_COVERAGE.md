## Source Standard

- AP CSP primary source: College Board AP Computer Science Principles course and exam information.
- AP CSA primary source: College Board AP Computer Science A course information and the current four-unit framework.
- AP CSA requested support source: CSAwesome2 AP CSA Java 2025+ from Runestone Academy.
- Java API support: Oracle Java documentation.
- Python support for AP CSP examples: Python Software Foundation documentation.
- All notes are original student-friendly summaries and do not copy source text verbatim.

## Student Note Structure

Every AP CSP and AP CSA lesson includes:

- Lesson title
- Learning objectives
- Key vocabulary
- Essential concepts
- Step-by-step explanation
- Worked example
- Code, pseudocode, diagram, or artifact example where useful
- Common mistakes
- Exam tips
- Summary
- Practice questions
- References

## Important AP CSA Curriculum Note

The AP CSA portal uses the latest College Board four-unit structure:

- Unit 1: Using Objects and Methods
- Unit 2: Selection and Iteration
- Unit 3: Class Creation
- Unit 4: Data Collections

The old ten-unit structure is not used.
